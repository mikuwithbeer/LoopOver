
 [ DESCRIPTION ]

   LoopOver is a lightweight CLI utility designed to reset the evaluation
   period for CrossOver on macOS. It programmatically modifies the
   application's property list to update the first run date,
   causing to refresh the trial cycle.


 ==============================================================================

 [ REQUIREMENTS ]

   * macOS (Requires CoreFoundation framework)
   * CrossOver installed in default location


 ==============================================================================

 [ USAGE ]

   Run the binary from your terminal. Sudo is usually not required as
   preference files typically belong to the user.
